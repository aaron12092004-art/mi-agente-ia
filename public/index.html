📱 Integración con WhatsApp
WhatsApp Business API requiere verificación de Meta. Seguí estos pasos para conectarlo.
📋 Pasos para conectar WhatsApp
  1. Creá una cuenta en Meta Developers
  2. Creá una App y habilitá WhatsApp Business API
  3. Obtendrás un Phone Number ID y un Token de Acceso
  4. Configurá el Webhook en Meta apuntando a tu servidor:
http://localhost:3000/api/whatsapp/webhook
  1. Pegá tus credenciales en el archivo .env
  2. Reiniciá el servidor
📝 Variables de Entorno Necesarias
WHATSAPP_TOKEN=tu_token_de_acceso
WHATSAPP_PHONE_NUMBER_ID=tu_phone_number_id
WHATSAPP_VERIFY_TOKEN=mi_token_secreto
